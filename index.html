Getting Started

Select a method: Direct Userscript? Bookmarklet?

Direct
  1. Copy the bookmarklet's link.
  2. Navigate to the the online store.
  3. Paste into your browser's address bar.
  4. Ensure the pasted line begins with javascript: If not, add it.
  5. Submit.
  6. PSDLE should activate.
Userscript
  1. Install Greasemonkey (Firefox), Tampermonkey (Chrome), or your browser's equivalent.
  2. Install the PSDLE userscript
  3. Navigate to the the online store.
  4. PSDLE should activate automatically.
Bookmarklet (Recommended for mobile)
  1. Get the bookmark.
  2. Navigate to the the online store.
  3. Select the bookmark from your bookmarks.
  4. PSDLE should activate.
After successful activation the PSDLE logo will appear in the bottom-left corner, click it to start.

The latest and (hopefully) greatest versions of PSDLE can be found in the repository. Use repository versions at your own risk.
Legacy versions are available here. Bug reports will not be accepted and support will not be given for these.

P.A.Q. (Potentially Asked Questions)

Is this safe or a ploy to phish account details?
Yes it is safe, no it is not for phishing account details.
I put it on GitHub for a reason, feel free to view the horribly effective code yourself and decide.
If/when I get pull requests I'll let them sit and look through them while allowing the "community" to as well.

If authentication details were left around openly on the download list page I would take my concerns elsewhere.

Can it go any faster?
Disabling "Catalog" during startup will significantly decrease the time between startup and the list at the cost of less features.

Does the search box support regex?
Yes! Simply wrap the regex in /, like usual.
PSDLE supports the /i modifier and a custom /d modifier which excludes matches.
For more information about Javascript Regex, click here.

Does this differentiate between Vita and PS/Vita TV games?
At the time of writing PSDLE can attempt to detect PS TV content for en-us only.
The option will only show up if using the en-us web store, which accepts only en-us accounts.
This is a limitation of the web store, not PSDLE.

If detected, "TV" will appear next to content's platform.
